```sgr cloud add [OPTIONS] DOMAIN_NAME```

Add a remote Splitgraph registry to .sgconfig with default parameters

### Options

  * **`--remote TEXT`**: Name for the remote (infer from the domain name by default)
  * **`-s, --skip-inject`**: Don't try to copy the config into all engines
