```sgr cloud plugins [OPTIONS] ```

List all Splitgraph Cloud data plugins.

This command lists all plugins available on a remote Splitgraph Cloud instance.

### Options

  * **`--remote TEXT`**: Name of the remote registry to use.
  * **`-f, --filter TEXT`**: Filter for plugins with this string in name/description
