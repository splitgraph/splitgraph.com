```sgr cloud seed [OPTIONS] SEED [DIRECTORY]```

Generate a starter Splitgraph Cloud project from a seed.

### Options

  * **`--remote TEXT`**: Name of the remote registry to use.
  * **`--github-repository TEXT`**: Name of the GitHub repository
